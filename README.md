# [When Last Login - Welcome Email Add On](https://yoohooplugins.com/plugins/when-last-login-welcome-email/)
![License](https://img.shields.io/badge/license-GPL--2.0%2B-blue.svg?style=flat-square)

## Welcome to the When Last Login - Welcome Email GitHub Repository

[When Last Login - Welcome Email](https://wordpress.org/plugins/when-last-login-welcome-email-add-on) extends the When Last Login plugin by sending a customizable HTML welcome email to users when they log in for the first time.

Customize the email subject, body content, logo, and footer credits. Use shortcode tags to personalize messages with the user's name and site details.

### Key Features

- **Customizable Emails** — Set your own subject line, body content, and footer credits.
- **Logo Support** — Add your brand logo to welcome emails via the WordPress media uploader.
- **Template Tags** — Use `**name**`, `**site_name**`, `**website_name**`, `**mail_subject**`, and `**footer_credit**` to personalize emails.
- **HTML Emails** — Send beautifully formatted HTML emails with responsive design.
- **Easy Integration** — Adds a settings tab directly to the When Last Login settings page.

## Installation

1. Install and activate the [When Last Login](https://wordpress.org/plugins/when-last-login/) plugin first.
2. Download the current development ZIP file: `https://github.com/Yoohoo-Plugins/when-last-login-welcome-email/archive/dev.zip`
3. Upload to `/wp-content/plugins/when-last-login-welcome-email/`
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Navigate to **When Last Login > Welcome Emails** to configure settings.

**Note:** Remove `-dev` from the folder name if installing from the development branch.

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
- [When Last Login](https://wordpress.org/plugins/when-last-login/) plugin (version 1.0.0 or higher)
